

   ***   Minicurso Jupyter Notebook - Jupyter, Python & SQL   ***   
           Laboratório Interinstitucional de e-Astronomia


Seja bem vindo! Seja bem vinda! 

Este minicurso faz parte do curso básico de ferramentas computacionais para e-astronomia oferecido pelo LIneA, que é composto por 3 minicursos independentes:

    1. LIneA Science Server
    2. Jupyter Notebook - Jupyter, Python & SQL (nível básico, este curso)
    3. Jupyter Notebook - Aplicações na Astronomia (em breve)

Todo o material didático utilizado nas aulas deste minicurso está disponível no repositório do curso no GitHub: 

    https://github.com/linea-it/curso.git 

Para acessá-lo, abra um Terminal (vá no menu "+" e selecione Terminal em "Other"). No Terminal, entre com o comando: 

    git clone https://github.com/linea-it/curso.git

Este comando vai criar uma pasta "curso" com uma cópia de todos os notebooks que serão utilizados nas aulas. Esta é a sua cópia, ou o seu "clone". Fique à vontade para editar e explorar os notebooks disponíveis. Caso encontre alguma dificuldade, peça ajuda por um dos nossos canais de comunicação:

    Slack: minicursoslinea.slack.com 
    e-mail: helpdesk@linea.gov.br


Bons estudos!
